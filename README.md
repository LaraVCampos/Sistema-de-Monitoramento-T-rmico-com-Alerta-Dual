# Sistema-de-Monitoramento-T-rmico-com-Alerta-Dual
Sistema de Monitoramento Térmico com Alerta Dual.Projeto embarcado em Arduino Uno para medição de temperatura em tempo real com o sensor LM35. Conta com sinalização visual por LEDs (Seguro, Atenção e Crítico) e alarme sonoro temporizado não-bloqueante. Desenvolvido para CPDs, estufas e logística farmacêutica.
